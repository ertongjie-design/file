##### 获取某个dom元素的内部属性整体

```html
<div id="hello"></div>
```

```js
var dv = document.getElementById("hello")//dv是<div id="hello"></div>
var dva = dv.attributes["id"]//dva是id="hello"
dva.nodeName//是"id"
dva.nodeValue//是"hello"
```
